# CovidChart
CovidChart projemde api ile rastgele sabit olan 5 şehirime covid sayısı oluşturup ui tarafında apiden gelen verileri grafik hale çevirdim.

## Teknolojiler
Projemde SignalR,Entityframework,Sql teknolojilerini kullandım.
